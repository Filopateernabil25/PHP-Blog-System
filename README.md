## Overview

PHP Blog System is an educational blog platform built using PHP, MySQL, and Bootstrap.  
The project is designed as a practical workshop that helps students understand real-world backend development concepts by building a complete blog system from scratch.

Through this project, students learn how to design databases, perform CRUD operations, manage content through an admin dashboard, handle file uploads, and build a responsive frontend interface.

## Features

- Responsive design using Bootstrap
- Blog posts management (Create, Read, Update, Delete)
- Category management system
- Secure admin dashboard
- Image upload handling
- Form validation
- Clean and simple user interface

## Learning Objectives

By the end of this workshop, students will be able to:

- Design and implement a relational database
- Perform CRUD operations using PHP & MySQL
- Create a secure admin dashboard
- Handle file uploads and form validation
- Build a responsive frontend with Bootstrap

## Tech Stack

- Backend: PHP
- Frontend: Bootstrap 5
- Database: MySQL
- Server: Apache (XAMPP)

## Installation

### Prerequisites

- PHP
- MySQL
- Apache Server

### Steps

1. Clone the repository
2. Move the project to the server directory (htdocs)
3. Create a database and import the SQL file
4. Configure database connection
5. Run the project on localhost

## Usage

- Browse blog posts
- View posts by category
- Access admin panel
- Add, edit, and delete posts
- Manage categories and images
